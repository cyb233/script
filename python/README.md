# Python 脚本

## 使用说明

1. 本目录收录各类 Python 脚本，请先准备好对应的 Python 运行环境。
2. 运行前请根据脚本内容安装所需依赖，并按本地环境调整网络、代理或配置参数。
3. 如脚本会生成输出文件，请留意当前工作目录或脚本内指定的输出路径。
